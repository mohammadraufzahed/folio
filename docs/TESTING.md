# Testing Guide for Folio PDF

## Running Tests

### Run All Tests
```bash
./vendor/bin/phpunit
```

### Run Specific Test File
```bash
./vendor/bin/phpunit tests/Styling/ColorTest.php
```

### Run Specific Test Method
```bash
./vendor/bin/phpunit --filter testRgb
```

### Run with Coverage
```bash
./vendor/bin/phpunit --coverage-html coverage
```

## Test Structure

```
tests/
├── Styling/
│   ├── ColorTest.php          # Color class tests
│   └── StyleTest.php          # Style class tests
├── Nodes/
│   └── PageTest.php           # Page node tests
└── Template/
    └── LexerTest.php          # Template lexer tests
```

## Writing New Tests

### Example Test Structure

```php
<?php

declare(strict_types=1);

namespace Folio\Pdf\Tests\YourNamespace;

use PHPUnit\Framework\TestCase;
use Folio\Pdf\YourClass;

final class YourClassTest extends TestCase
{
    public function testSomething(): void
    {
        $instance = new YourClass();
        $result = $instance->method();
        
        $this->assertEquals('expected', $result);
    }
}
```

## Testing the PDF Engine End-to-End

### Create a Test Script

Create `test-generation.php`:

```php
<?php

require_once __DIR__ . '/vendor/autoload.php';

use Folio\Pdf\Document\Pdf;
use Folio\Pdf\Nodes\Column;
use Folio\Pdf\Nodes\Heading;
use Folio\Pdf\Nodes\Page;
use Folio\Pdf\Nodes\Text;
use Folio\Pdf\Styling\Color;
use Folio\Pdf\Styling\Style;

// Test basic PDF generation
Pdf::make()
    ->page(
        Page::a4()->withContent(
            Column::make()
                ->addChildren([
                    Heading::h1('Test Document'),
                    Text::make('This is a test PDF generated by Folio PDF.'),
                ])
        )
    )
    ->save(__DIR__ . '/test-output.pdf');

echo "Test PDF generated: test-output.pdf\n";
```

### Run the Test Script
```bash
php test-generation.php
```

## Common Testing Patterns

### Testing Immutable Objects
```php
public function testImmutability(): void
{
    $original = Style::make();
    $modified = $original->withPadding(10.0);
    
    $this->assertNotSame($original, $modified);
    $this->assertNull($original->padding());
    $this->assertEquals(10.0, $modified->padding());
}
```

### Testing Fluent Interfaces
```php
public function testMethodChaining(): void
{
    $style = Style::make()
        ->withPadding(10.0)
        ->withMargin(20.0)
        ->withColor(Color::black());
    
    $this->assertEquals(10.0, $style->padding());
    $this->assertEquals(20.0, $style->margin());
}
```

### Testing Edge Cases
```php
public function testEdgeCases(): void
{
    // Test with zero values
    $color = Color::rgb(0, 0, 0);
    $this->assertEquals(0.0, $color->red());
    
    // Test with maximum values
    $color = Color::rgb(255, 255, 255);
    $this->assertEquals(1.0, $color->red());
    
    // Test clamping
    $color = Color::rgb(300, -50, 0);
    $this->assertEquals(1.0, $color->red());
    $this->assertEquals(0.0, $color->green());
}
```

## Continuous Integration

### GitHub Actions Example

```yaml
name: Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Setup PHP
      uses: shivammathur/setup-php@v2
      with:
        php-version: '8.3'
    
    - name: Install dependencies
      run: composer install --prefer-dist --no-progress
    
    - name: Run tests
      run: ./vendor/bin/phpunit
    
    - name: Run static analysis
      run: ./vendor/bin/phpstan analyse src --level=8
```

```bash
./vendor/bin/phpunit --stop-on-failure
```

### Run specific test with debugging
```bash
./vendor/bin/phpunit --filter testMethodName --debug
```

## Performance Testing

### Benchmark PDF Generation

```php
<?php

$start = microtime(true);

for ($i = 0; $i < 100; $i++) {
    Pdf::make()
        ->page(Page::a4()->withContent(
            Column::make()->addChildren([
                Text::make("Page $i")
            ])
        ))
        ->save("/tmp/bench-$i.pdf");
}

$end = microtime(true);
echo "Generated 100 PDFs in " . ($end - $start) . " seconds\n";
```

## Integration Testing

### Test Template Compilation

```php
<?php

use Folio\Pdf\Template\PhpTemplateCompiler;

$compiler = new PhpTemplateCompiler();
$template = 'page { column { heading "Test" } }';

$phpCode = $compiler->compile($template);
echo $phpCode; // Output generated PHP code
```

This will help you understand how the template compiler transforms your templates into executable PHP code.
